"# basic-aam" 

The work here explores the implementation of the VOSM AAM Library. We have used the HELEN dataset hosted by iBug. 
To execute the program run the ConsoleApplication1.cpp 

System Requirements::
1) OpenCV 2.4.3
2) Visual Studio 2013
